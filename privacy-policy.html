1. מהי OrdexAI ועל מי חלה מדיניות זו

OrdexAI היא מערכת B2B לניהול הזמנות דרך WhatsApp, המיועדת למפיצי מזון. המערכת מאפשרת ללקוחות העסקיים של המפיץ לשלוח הזמנות בשפה חופשית — טקסט, קול או תמונה — ישירות בוואטסאפ, ומעבדת אותן אוטומטית באמצעות בינה מלאכותית.

מדיניות זו חלה על:

  • מפיצים — עסקים הנרשמים ל-OrdexAI ומשתמשים במערכת לקבלת הזמנות.
  • לקוחות המפיץ — עסקים המזמינים מוצרים דרך WhatsApp.
  • משתמשי דף ההזמנות — מי שניגש לדף ההזמנה המקוון בכתובת order.ordexai.com.
OrdexAI פועלת כמעבד נתונים (Data Processor) עבור המפיץ, אשר הוא בעל הנתונים (Data Controller) מול לקוחותיו.

3. כיצד המערכת מעבדת הזמנות

כאשר לקוח שולח הזמנה:

  • טקסט: תוכן ההודעה נשלח ל-Claude (Anthropic) לניתוח שפה טבעית וחילוץ פריטים וכמויות.
  • קול: קובץ השמע נשלח ל-Soniox לתמלול בעברית, ולאחר מכן הטקסט המתומלל מעובד כמו הזמנת טקסט. הקובץ המקורי אינו נשמר.
  • תמונה / טופס: התמונה נשלחת ל-Azure Document Intelligence לחילוץ טקסט (OCR), ולאחר מכן הטקסט מעובד כמו הזמנת טקסט. התמונה המקורית אינה נשמרת.
  • תוצאות ההזמנה המאושרת נשמרות במסד הנתונים ומרוכזות ברשימת פיקינג PDF הנשלחת למפיץ בסוף כל יום דרך WhatsApp ואימייל.

Privacy Policy

Last updated: June 2026 | OrdexAI — Operated by Lishai Shimon Ben Shoshan

1. What is OrdexAI and who this policy applies to

OrdexAI is a B2B WhatsApp-based order management system for food distributors. The system enables a distributor's business customers to send orders in free-form language — text, voice, or photos — directly via WhatsApp, which are then processed automatically using AI.

This policy applies to:

  • Distributors — businesses that sign up to OrdexAI and use the system to receive orders.
  • Distributor's customers — businesses that place orders via WhatsApp.
  • Order page users — anyone accessing the online ordering page at order.ordexai.com.
OrdexAI acts as a Data Processor on behalf of the distributor, who is the Data Controller with respect to their customers' data.

3. How the system processes orders

When a customer sends an order:

  • Text: The message content is sent to Claude (Anthropic) for natural language processing and extraction of items and quantities.
  • Voice: The audio file is sent to Soniox for Hebrew transcription, after which the transcribed text is processed like a text order. The original audio file is not retained.
  • Photo / Form: The image is sent to Azure Document Intelligence for OCR text extraction, after which the text is processed like a text order. The original image is not retained.
  • Confirmed order results are stored in the database and compiled into a daily PDF picking list sent to the distributor via WhatsApp and email.

5. Data storage and retention

  • Order data is retained for ongoing operations and system improvement.
  • Audio files and images are not retained — processed and deleted.
  • Picking list PDFs are generated in-memory and are not stored on servers.
  • The database is hosted on AWS in the EU (Frankfurt) region and protected with encryption.

6. Data security

We implement security measures including: encrypted communications (HTTPS/TLS), HMAC verification on all WhatsApp messages, per-organization access controls (Row-Level Security), and API keys protected via AWS SSM. No system is 100% secure.

7. Your rights

  • To review the data stored about you.
  • To request correction or deletion of your data.
  • Opt-out: Reply "STOP" (or "הסר") to any WhatsApp message, or email info@ordexai.com.
  • Distributor customers may also contact their distributor directly regarding their data.
Audio files and images are processed but never stored. Only the extracted text is retained as part of the order record.
